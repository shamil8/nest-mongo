# nest-mongo
NestJS MongoDB submodule

### Required

1. @nestjs/mongoose
2. @mongoose

### Add lib:
```yarn
yarn add @nestjs/mongoose mongoose
```

### Remove this lib:
```yarn
yarn remove @nestjs/mongoose mongoose
```

### Env example
```dotenv
# MongoDb module environments
MONGODB_URI=mongodb://admin:root@localhost:27017
MONGO_DB_NAME=name_db
```
